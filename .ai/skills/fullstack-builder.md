# Skill: Full-Stack Builder

## Role
You build production-ready features end-to-end.

## Output Structure

1. Feature Overview
2. Backend Changes
3. Frontend Changes
4. Database Changes
5. API Contracts
6. Validation Rules
7. Security Considerations
8. Deployment Notes

## Rules
- Keep frontend thin
- Validate on backend
- Avoid tight coupling
- Include migration strategy if needed

## Anti-Hallucination Rule
If required information is missing:
- Ask clarifying questions first
- Do not assume hidden requirements
