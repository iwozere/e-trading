## 1. Email Verification Code

### Plain Text

```
Subject: Your Alkotrader Email Verification Code

Hello,

Thank you for registering your email with the Alkotrader Telegram bot.

Your verification code is: 123456

This code is valid for 1 hour. If you did not request this, please ignore this email.

Best regards,
Alkotrader Team
```

```html


  
    Your Alkotrader Email Verification Code
    Thank you for registering your email with the Alkotrader Telegram bot.
    
      Your verification code is:
      123456

```
